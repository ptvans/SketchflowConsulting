# Archive: Sketchflow Consulting Website

This directory contains the original Sketchflow Consulting static landing page,
archived when the repo was repurposed to build the Pivot career exploration app.

## Original Stack
- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Express.js backend
- Framer Motion animations

## Original Purpose
A consulting/portfolio site for Paul's product design consulting business.
